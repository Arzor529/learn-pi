Welcome to Learn Pi!

Learn Pi offers a range of simple and exciting Raspberry Pi based computing projects for students aged 12-14.
Learn about hardware, software, and programming to create exciting games and projects!

What does Learn Pi Offer?

Learn Pi offers various exciting features including:

  • Learning to set up and use a Raspberry Pi!
  • Using programming to change the way you play Minecraft!
  • Creating your very own game using Scratch!
  • You can add your friends to share the Learn Pi Experience!
  • Earning Rewards that will appear on your profile!
  • You can place your custom games on your profile for friends to play!
